# Small Business Website Delivery Workflow

This project models a real-world workflow for delivering static websites to small businesses using cloud-based tools. It focuses on requirement intake, backend processing, and clear delivery steps rather than frontend complexity.

## Project Goal

To demonstrate how a small business website request can be collected, processed, and managed using a lightweight cloud backend.

## Planned Features

- Client website request form
- Serverless backend to receive and store requests
- Cloud-first, cost-efficient architecture
- Clear documentation of the delivery process

## Tech Stack (Planned)

- Frontend: HTML, CSS, JavaScript
- Backend: Azure Functions (HTTP-triggered)
- Storage: Azure Storage (Table or Blob)
- Version Control: Git & GitHub
